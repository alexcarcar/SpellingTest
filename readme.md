# 📚 Spelling Test Android App

An interactive Android app designed to help students practice spelling through self-guided tests. Perfect for parents and educators looking for a simple, effective way to reinforce weekly spelling lists.

---

## ✨ Features

- 🔤 **Custom Word Lists**: Parents or students can enter their own spelling words.
- 🔊 **Voice Playback**: The app reads each word aloud using text-to-speech.
- ⌨️ **Typing Practice**: Students type in the correct spelling for each word.
- 🔁 **Retesting Made Easy**: Update word lists weekly to match classroom assignments.
- 🧑‍🎓 **Independent Learning**: Ideal for unsupervised practice at home.
